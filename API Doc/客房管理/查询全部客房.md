# 查询全部客房

查询全部客房的信息（不包含入住情况）

## route

`/api/room/show_all`

## params

| key | description | type | example |
|:---:|:---:|:---:|:---:|


## Result

列表，每个元素为一个房间

| key | description | type | example | 
|:---:|:---:|:---:|:---:| 
|id|房间号|str|A105|
|type|房间类型|str|	“温馨大床房”|
|floor|楼层|int|1|
|desc|房间描述|str|"..."|
|std_price|标准价格|double|100.5|
